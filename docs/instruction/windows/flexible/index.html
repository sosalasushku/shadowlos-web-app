Для проксирования приложений
Для проксирования целых доменов
1
Перейдите в Настройки, затем во вкладку Базовые маршруты. Внизу в пункте Outbound по умолчанию выберите bypass. После этого нажмите Ок.
2
Снова откройте Настройки и перейдите в раздел TUN-режим. В появившемся окне в колонке Проксировать процессы укажите приложения, которые должны использоваться с VPN. Правильные названия процессов можно найти в Диспетчере задач в свойствах соответствующего процесса.
3
Проверьте, что внизу в пункте Режим белого списка стоит галочка. После этого нажмите Ок, а затем закройте всплывшее сообщение, нажав Ок.
4
Запустите конфиг. Теперь трафик идет только через выбранные вами приложения.
1
Перейдите в Настройки, затем во вкладку Базовые маршруты и выберите внизу слева пункт Кастомные маршруты.
2

3
