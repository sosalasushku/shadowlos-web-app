Android
⚙️ Гибкий
1
Скачайте клиент V2RayTun:
Скачать из Google Play
2
Сохраните себе на устройство файл конфигурации:
Сохранить Config.json
3
Импортируйте файл Config.json в V2RayTun.
4
Нажмите на пункт Маршрутизация, далее на Маршрутизация выбранных приложений.
5
Выберите приложения, которые вы хотите использовать с VPN, остальные приложения будут работать без проксирования. Для удобства вы можете найти нужное вам приложение через поиск, для этого нажмите на значок лупы справа вверху и введите название искомого приложения.
6
На странице Подключение выберите пункт Сервисы и укажите доменные зоны, которые вы хотите исключить из использования VPN. Рекомендуем добавить зону .RU.
7
Теперь VPN будет работать только в выбранных вами приложениях.
