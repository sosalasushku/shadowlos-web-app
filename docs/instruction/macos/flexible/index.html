Для исключения проксирования сайтов

1
Нажмите правой кнопкой на значок V2rayU на панели задач и перейдите в Routing и выберите пункт Routing…. Скопируйте ниже образец для исключения сайтов и вставьте его в поле Direct в открывшемся окне. По образцу впишите сайты, которые вы не хотите проксировать.
